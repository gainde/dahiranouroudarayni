Options

Dahira

Cette fenêtre permet de mettre à jour les informations de la dahira. Cliquer sur Éditer pour modifier les informations de la dahira.

Membre

Ajouter un nouveau membre

Pour ajouter un nouveau membre il faut aller dans la fenètre principale , cliquer sur : Membre –> Ajouter ; Une fenêtre s’ouvre et dans celle-ci entrer les informations du nouveau membre puis cliquer sur l’un des boutons, Enregistrer pour enregistrer ou Annuler pour annuler l’ajout du nouveau membre.

Editer un membre

Pour éditer un membre il faut aller dans la fenètre principale , cliquer sur : Membre , sélecionner le membre à éditer dans la liste des membres en cliquant sur celle –ci; Et le bouton Editer sera actif, ensuite cliquer sur ce bouton; Une fenêtre s’ouvre et dans celle-ci entrer les informations du membre puis cliquer sur l’un des boutons, Enregistrer pour enregistrer ou Annuler pour annuler les modifications apportées.

Supprimer un membre

Pour supprimer un membre il faut aller dans la fenètre principale , cliquer sur : Membre , sélecionner le membre à supprimer dans la liste des membres en cliquant sur celle –ci; Et le bouton Supprimer sera actif, ensuite cliquer sur ce bouton.

Ajouter cotisation Loyer , KST ou évènement d’un membre

Pour ajouter une cotisation d’un membre il faut aller dans la fenètre principale , cliquer sur : Membre , sélecionner le membre dans la liste des membres en cliquant sur celle –ci; Et le bouton Cotisation sera actif, ensuite cliquer sur ce bouton; Une fenêtre s’ouvre et dans celle-ci entrer le montant à cotiser pour ce membre suivant le type de cotisation (Loyer,KST ou Évènement)puis cliquer sur le bouton Ajouter.

Envoyer le relevé d’impôt pour un membre

Pour envoyer le relevé d’impôt pour un membre il faut aller dans la fenètre principale , cliquer sur : Membre , sélecionner le membre dans la liste des membres en cliquant sur celle –ci; Et le bouton Impôt sera actif, ensuite cliquer sur ce bouton; Une fenêtre s’ouvre et dans celle-ci entrer les informations nécessaires puis cliquer sur le bouton Envoyer.

Évènement

Ajouter un évènement

Pour ajouter un nouvel évènement il faut aller dans la fenètre principale , cliquer sur : Évènement –> Nouvel Évènement; Entrer les informations puis cliquer sur le bouton Ajouter.

Éditer un évènement

Pour éditer un évènement il faut aller dans la fenètre principale , cliquer sur : Évènement, sélecionner l’évènement à éditer dans la liste des évènements en cliquant sur celle –ci; Et le bouton Editer sera actif, ensuite cliquer sur ce bouton; Entrer les informations puis cliquer sur le bouton Enregistrer ou Annuler.

Supprimer un évènement

Pour supprimer un évènement il faut aller dans la fenètre principale , cliquer sur : Évènement, sélecionner l’évènement à supprimer dans la liste des évènements en cliquant sur celle –ci; Et le bouton Supprimer sera actif, ensuite cliquer sur ce bouton;

Impôt

Pour envoyer les relevés d’impôt il faut aller dans la fenêtre principale, cliquer sur : Impôt -> Parcourir; Choisir le dossier pour enregistrer les pdf des impots pour chaque membre; Ensuite entrer les informations nécessaires puis cliquer sur le bouton Executer.
